# hexagonal-game-of-life

<a href="http://aybberk.github.io/hexagonal-game-of-life"> Conway's Game of Life with Hexagonal Grid on HTML canvas. </a>

<h1>Features</h1>
<ul>  
  <li> Mouse toggle </li>
  <li> Pause, clear etc. </li>  
  <li> Interactive rules. </li>
</ul>
